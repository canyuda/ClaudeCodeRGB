# ESP32-C3 RGB Case Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a pure Python generator that exports one STL file containing the bottom tray and lid for the ESP32-C3 RGB enclosure.

**Architecture:** Use simple parametric geometry and emit binary STL triangles directly, avoiding unavailable CAD dependencies. Model the tray from cuboid wall segments and the lid from a tiled plate that omits cells for circular and rectangular openings.

**Tech Stack:** Python 3 standard library, unittest.
